# Filtrado IIR para comunicaciones satelitales – UTN FRBA

Desarrollo de filtros IIR Chebyshev en Python para mejorar la claridad 
de voz en canales satelitales afectados por interferencias y ruido espacial.

## Descripción
- Implementación de filtros Chebyshev paso alto y paso bajo.
- Modelado del canal con interferencias y ruido espacial.
- Análisis espectral y validación del desempeño sobre señales reales.

## Herramientas
Python · NumPy · SciPy · Matplotlib · Google Colab
